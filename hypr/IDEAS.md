# Hyprland Ideas

Ideas utiles para considerar mas adelante. No estan implementadas.

## Flujo diario

- `mainMod + Return` para abrir terminal rapido.
- `mainMod + Space` para `togglefloating`.
- `mainMod + F` para fullscreen.
- `mainMod + Shift + R` para recargar Hyprland.
- `mainMod + Escape` para bloquear pantalla con `hyprlock`.

## Ventanas

- Familia de binds para mover ventanas con teclado.
- Familia de binds para redimensionar ventanas con teclado.
- Un atajo para centrar o fijar una ventana flotante.
- Recuperar scratchpad con otra tecla que no choque con screenshots.

## Multimedia

- OSD/notificaciones visuales para volumen, brillo y microfono.
- Bind para mute de salida separado del mute del microfono.
- Binds para cambiar dispositivo de audio si alguna vez usas varios.

## Screenshots

- Bind extra para captura de pantalla completa.
- Bind para abrir directamente la ultima captura.
- Integracion con una herramienta de anotaciones si algun dia la necesitas.

## Sesion y sistema

- Menu rapido de apagar, reiniciar, suspender y cerrar sesion.
- Bind para reiniciar Waybar cuando cambies su config.
- Bind para cambiar wallpaper o rotarlo.

## Waybar

- Mostrar estado de VPN o red mas detallado.
- Mostrar layout de teclado si empiezas a cambiar entre varios.
- Agregar iconos o estados para bluetooth si lo usas seguido.
